## Ben-vindo ao Store
Store é um projeto simples usado para terinar o crud com as rotas HTTP, 
usando uma api criada com Json-Server

## Começando
* Faça um clone ou baixar o arquivo zip
* instale as dependencias:
~~~javascript
    $ npm install
    //ou
    $ yarn
~~~
* Inicie o projeto:
~~~javascript
    $ npm start
    //ou
    $ yarn start
~~~
